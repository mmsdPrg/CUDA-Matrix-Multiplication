# HW4 - CUDA Matrix Multiplication

پروژه بهینه‌سازی ضرب ماتریس با CUDA برای شبکه‌های عصبی

## ساختار پروژه

```
├── matmul_*.cu              # کرنل‌های CUDA
├── train.py                 # اسکریپت آموزش
├── analyze_results.py       # تحلیل نتایج
├── Makefile                 # اتوماسیون
└── report.html              # گزارش نهایی
```

## اجرا

### لینوکس
```bash
make run-all
```

### ویندوز
```bash
RUN_ALL.bat
```

## نتایج

نتایج در پوشه `results/` ذخیره می‌شوند.

گزارش کامل در فایل `report.html` موجود است.
